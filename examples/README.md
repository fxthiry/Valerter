# Examples

Real-world examples showing how to configure Valerter for different use cases.

Each example demonstrates the same 4-step pattern:
1. **Query** — Filter logs with LogsQL
2. **Parser** — Extract fields (JSON + regex)
3. **Throttle** — Prevent alert spam
4. **Template** — Format notifications

## Available Examples

| Example | Description |
|---------|-------------|
