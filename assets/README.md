This repository folder contains the official Valerter mark in monochrome variants.

## Files

- `svg/valerter-mark-standard.svg`  
  Monochrome with opacity levels (best general use on light backgrounds).

- `svg/valerter-mark-onecolor.svg`  
  Single-color version using `currentColor` (ideal for CSS theming, print, stamps).

- `svg/valerter-mark-inverted.svg`  
  White mark (best on dark backgrounds).

- `svg/valerter-lockup-standard.svg`  
  Icon + "Valerter" wordmark. The wordmark uses a font stack (Inter preferred).  
  For pixel-perfect brand assets, convert text to outlines in your design tool.

## Recommended usage

- **Product icon / favicon**: use `valerter-mark-*`
- **Docs / website header**: use the lockup (`valerter-lockup-standard.svg`)
- **Backgrounds**
  - Light background: `standard`
  - Dark background: `inverted`

## Color

Default monochrome:
- Primary: `#111827` (near-black)

## Sizing

- Prefer using SVG whenever possible.
- For PNG usage, choose the closest size >= your target and let the UI downscale.
